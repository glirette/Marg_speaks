# Marg Speaks Case Archive

This repository is intentionally preserved as an early public-record and New Orleans-area crime documentation artifact by Greg Lirette.

The original page, [default.htm](default.htm), was created from a Microsoft Word HTML export and used GitHub as one experimental publication channel. GitHub did not become the live publishing surface for this coverage. Greg continued following the case elsewhere, but this repository itself was not kept current at the time.

That is now documented plainly so the repository does not look abandoned by accident.

## Current Status

As of this refresh on 2026-06-08:

- Terry Christopher Speaks was convicted in Jefferson Parish of second degree murder, obstruction of justice, and conspiracy to obstruct justice in the death of Jaren Lockhart. The Louisiana Fifth Circuit Court of Appeal affirmed his convictions and sentences in 2016.
- Margaret A. Sanchez pleaded guilty in the case and was sentenced to 40 years in prison, according to public reporting by WDSU.

See [updates/2026-06-08-case-status.md](updates/2026-06-08-case-status.md) for the cited status note.

Machine-readable status is available at [data/case-status-2026-06-08.json](data/case-status-2026-06-08.json).

## Why This Repo Exists

This repository matters less as a modern website and more as provenance.

It shows that Greg Lirette was following and documenting Orleans/Jefferson Parish crime, court process, public records, and accountability issues long before the current Notary Geek and public source-quality work. The original file remains part of that record.

Future public crime/court repositories may be more structured and current. This one is being preserved and refreshed without rewriting the old artifact.

## Original Artifact

- [default.htm](default.htm) is preserved as the original GitHub-published HTML page.
- [default_files/](default_files/) contains support files generated by the original export.
- The original tracked files are intentionally left in place, including old Microsoft Office export artifacts.

## Limits

This repository is not an official court record, legal filing, or active news site. For current legal status, use court records and cited public sources.
