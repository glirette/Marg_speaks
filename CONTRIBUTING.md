# Contributing

This repository preserves an early public crime/court documentation artifact and accepts source-backed corrections or additions.

## Source Standard

Contributions should be tied to public, checkable sources such as:

- court opinions
- docket records
- sentencing records
- archived public reporting
- official agency records
- contemporaneous public documents

Do not submit unsourced accusations, rumors, or private personal information.

## Language Standard

Keep legal status language precise:

- Use `charged` only when the source supports a charge.
- Use `convicted` only when the source supports a conviction.
- Use `pleaded guilty` only when the source supports a plea.
- Separate allegations, charges, convictions, sentencing, appeals, and later custody status.

## Precision and Corrections

Greg Lirette holds this work to a high personal standard for accuracy. If a note is based on memory, assumption, inference, or incomplete source access, say that plainly. If later records show that wording was wrong or imprecise, correct the repository rather than quietly leaving the record muddy.

The standard here is not to sound certain. The standard is to be clear about what is known, what is sourced, what is inferred, and what still needs verification.

## Original Archive

The original `default.htm` file should not be rewritten casually. New context should generally be added in `updates/`, `data/`, or future documentation files so the original artifact remains useful as provenance.
